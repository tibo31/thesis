# Thesis Applications

This repository provides supplementary RMarkdown documents and outputs 
related to the empirical applications presented in Thibault Laurent’s PhD thesis (2025).

## Structure
- `chapter_1/` : Migration flow estimation based on optimal transport.  
- `chapter_2/` : Climate indicator construction and analysis.  
- `chapter_3/` : Climate impacts on international migration flows  
  (application of the chapter *Impact decomposition for the SAR interaction model*).  
- `chapter_4/` : Climate impacts on migration compositions  
  (application of the chapter *Covariates impacts in SAR models for CoDa*).  

Each folder contains:
- `application_for_thesis.Rmd` – RMarkdown source code.  
- `application_for_thesis.html` – HTML output.  
- subdirectories for the reproduction of figures and tables from the corresponding chapter or article.
